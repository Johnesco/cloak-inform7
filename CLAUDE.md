# Cloak of Darkness

The canonical IF benchmark game, implemented in Inform 7. Originally designed by Roger Firth as a standard demonstration across interactive fiction systems. This version is from the official Inform 7 documentation by Graham Nelson and Emily Short.

For build, test, and publish workflows, see `C:\code\ifhub\reference\project-guide.md`.

## Game Overview

- **Max score**: 2 points (hang cloak + read message)
- **Rooms**: Foyer of the Opera House, Cloakroom, Foyer Bar
- **Key mechanics**: The velvet cloak makes the Bar dark when carried. Hang it on the hook in the Cloakroom to light the Bar. Fumbling in the dark degrades the message (neat -> scuffed -> trampled). Read the neat message to win.
